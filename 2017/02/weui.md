> weui [github](https://github.com/weui/weui)

weui是微信团队基于微信视觉体验开发的前端样式库，方便开发者快速的开发一套与微信客户端视觉一致的UI界面

### 基础组件

> flex

``` js
<div class="weui-flex">
	<div class="weui-flex__item"></div>
	<div class="weui-flex__item"></div>
	<div class="weui-flex__item"></div>
</div>
```
